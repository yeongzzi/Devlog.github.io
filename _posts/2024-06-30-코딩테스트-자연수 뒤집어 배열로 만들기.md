### 문제 설명

자연수 n을 뒤집어 각 자리 숫자를 원소로 가지는 배열 형태로 리턴해주세요. 예를들어 n이 12345이면 [5,4,3,2,1]을 리턴합니다.

### 나의 풀이

```jsx
function solution(n) {
  // 방법1
  return (n + "")
    .split("")
    .reverse()
    .map((e) => parseInt(e));
  // 방법2
  return [...(n + "")].reverse().map((e) => e * 1);
}
```
